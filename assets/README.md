# assets/

Reserved for future static assets.

Planned:
- `sounds/`   — Optional .ogg / .mp3 sound files (currently synthesised via Web Audio API in ui.js)
- `icons/`    — App icon / favicon (e.g. chess-icon.png)
- `pieces/`   — Optional SVG piece sets to replace Unicode glyphs

The application works fully without any files in this folder.
